# Candle native DLL

Place the x64 `candle.dll` from the Candle.NET/Cangaroo Candle API build in this directory.
When present, `CanMonitor.Wpf.csproj` copies it next to `CanMonitor.Wpf.exe`.

The current development machine does not have the Visual C++ build targets required to build
`candle.dll` from source automatically.
